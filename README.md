# 台股即時看板 (DailyStock)

這是一個輕量、極簡的台灣股票即時報價看板網頁。採用**完全純前端 (Pure SPA)** 架構，無須任何後端伺服器或安裝環境，只需一個檔案即可運作。

## 核心特色

*   **無伺服器架構 (Serverless)**：所有資料請求與邏輯皆於瀏覽器端執行。無需安裝 Node.js 或任何環境，只要點兩下開啟 `index.html` 即可使用。
*   **全市場支援**：透過 Fugle Market Data API，支援**上市 (tse)**、**上櫃 (otc)** 及 **興櫃 (esb)** 股票的即時報價。
*   **智慧分頁與效能極致**：
    *   內建分頁功能 (可選每頁 10, 15, 20, 25, 30 筆)，畫面乾淨俐落。
    *   **API 最佳化**：自動更新時「只抓取當前頁面」的股票報價，徹底解決 API 速率限制 (Rate Limit) 問題，更新速度極快。
*   **靈活的資料管理**：
    *   **UI 密鑰管理**：API Key 直接儲存於瀏覽器本地，免除 `.env` 設定煩惱，安全又方便。
    *   **一鍵備份還原**：提供「儲存記錄」與「匯入記錄」功能，無縫在不同電腦間移轉股票清單、交易記錄及各項自訂設定。
*   **自動化與客製化**：
    *   自動於交易時間內更新報價，收盤後自動停止更新以節省流量。
    *   支援一鍵切換自動更新頻率 (6秒、12秒、30秒)。
    *   自動附帶產業分類標籤（快取自 FinMind API）。
    *   點擊股票代號或名稱可直接開啟 Goodinfo 公司簡介頁面。

## 快速上手

1.  **取得 Fugle API Key**：
    請至 [Fugle 開發者中心](https://developer.fugle.tw/) 註冊並取得免費的 API Key。

2.  **開啟看板**：
    在任何電腦上，直接雙擊用瀏覽器（Chrome, Edge, Safari 等）打開本資料夾內的 `index.html` 檔案。

3.  **設定 API Key**：
    點擊畫面右下角的「🔑 API Key」按鈕，貼上您的 Fugle API Key 並按下確認。設定將安全地保存在您的瀏覽器中。

## 使用說明

1.  **新增/隱藏股票**：在左上角輸入股票代號（支援逗號分隔，如 `2330, 2454, 7886`），按下「Enter」或「立即新增」按鈕即可。若輸入已存在的代號，則會將其隱藏。
2.  **紀錄資料**：直接在表格中的「現存股數」或「交易記錄」欄位輸入文字，系統會自動在您打字的當下存入瀏覽器。
3.  **切換個人偏好**：您可以在畫面中隨時切換每頁顯示的股票筆數，或在右下角切換資料的更新頻率，這些個人偏好設定都會被自動記憶。
4.  **備份與移轉資料**：
    *   點擊右下角的「📤 儲存記錄」，系統會為您下載一份包含所有資料（股票清單、筆記、偏好設定）的 `.json` 備份檔（*不含 API Key 以策安全*）。
    *   未來更換電腦或瀏覽器時，只要點開 `index.html` 並點擊「📥 匯入記錄」，選擇該檔案，即可完美還原所有心血。

## 備用後端方案 (舊版架構)

為了防止未來可能遇到的跨網域 (CORS) 政策變更，本專案仍保留了舊版的 `server.js` 與 `package.json` 作為緊急備用代理伺服器方案。
（日常使用不需要理會這些檔案）
若有需要重啟後端：
1. 在目錄建立 `.env` 檔，並填入 `FUGLE_API_KEY=你的_KEY`。
2. 開啟終端機執行 `node server.js`。
3. 瀏覽器前往 `http://localhost:3000` 即可使用。
